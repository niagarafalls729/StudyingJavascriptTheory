## 📜 주요 학습 내용  

### 1. **JavaScript 기초**
- 변수 선언 (`var`, `let`, `const`)
- 데이터 타입 및 변환
- 스코프와 호이스팅
- 조건문과 반복문

### 2. **고급 개념**
- 클로저(Closure)와 실행 컨텍스트
- 비동기 JavaScript: Promises, Async/Await
- `this`와 바인딩
- ES6+의 주요 기능 (Arrow Functions, Destructuring 등)

### 3. **실전 예제**
- 이론을 활용한 실습 코드
- 일반적인 면접 질문과 답변 준비
